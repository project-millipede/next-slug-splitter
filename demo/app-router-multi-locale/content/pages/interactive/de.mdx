# Interaktive Demo

Dies ist eine **schwere** Seite — sie nutzt eine eigene React-Komponente, die
zusätzliches JavaScript an den Browser ausliefert.

## Live Counter

Der Counter unten ist eine zustandsbehaftete React-Komponente direkt in MDX:

<Counter />

Weil diese Seite `<Counter />` nutzt, stuft next-slug-splitter sie als
_schwere_ Route ein und generiert einen eigenen Handler, der nur die
`Counter`-Komponente importiert.

Leichte Seiten wie "Erste Schritte" bündeln diesen Komponenten-Code nicht.
